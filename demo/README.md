# demo

This is a demo front-end site for meteor-snap.

## Before start

Running the snap first if you want to test it locally, or edit `vite.config.ts` to change the `SNAP_ORIGIN` to the npm package.

## Install dependencies

```bash
pnpm install
```

## Running the demo

```bash
pnpm dev
```

## Building the demo

```bash
pnpm build
```
